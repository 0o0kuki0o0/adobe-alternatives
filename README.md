# Adobe Alternatives

> A list of alternatives for Adobe software, originally collected by [@XdanielArt](https://twitter.com/XdanielArt) and improved upon by the community,

**Legend:**
- ✨ = Free & open source
- ⭐️ = Free of charge
- 💵 = Single purchase
- 🔒 = Subscription for more functions
